# DeepNN1

A very minimalistic Neural Network library written in C#.

Supports Perceptron/Sigmoid Neurons.

The code is set in a way of a demonstration. It trains itself and shows MNIST digits recognition.

It does not depend on but currently uses Unity3D to render the demonstration.


## How to use

Load it on Unity and click run. =)


## Why, if we have so many powerful libraries already?

It was written several years ago, there were not so many of them yet. But I only decided to upload to GitHub today.

Also, I wrote it for learning, in practice, how an Artificial Neuron works internally.
